Covid 19 Death Rate Per State

Rationale for design decisions

We chose overlapping line graphs because it would be best to show quantitative data and how it rises over time.

We also wanted to be able to see a snapshot of all the states for a single day. So our interactive design included hovering over a certain date and the number of deaths for each state being displayed.

The visual encoding we chose were state as nominal, date as temporal and deaths was quantitative.

We chose state as nominal because states are simply labels of each category. This is what we want to categorize our data as.

We chose deaths as quantiative because it is a measured quantity that we want to compare for different states.

We chose date as temporal becuase the value of the variable is the last assigned value. This is always the case with date.

Alternatives

Initially we considered doing a heat map of the United States but we ran into troubles working with d3 so we went with vega-lite.
